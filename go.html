Go cơ bản

T4 24 Tháng 7 2019

Các kiểu dữ liệu trong go

  • Cơ bản: số (int, float, complex), chuỗi (string) và boolean
  • Tập hợp: mảng (array) và cấu trúc (struct)
  • Tham chiếu: pointer, slice, map, function và channel
  • Giao diện (interface)

Số nguyên, số thực và luận lý (boolean)

  • Số nguyên có dấu: int8, int16, int32, int64
  • Số nguyên không dấu: uint8, uint16, uint32, uint64
  • Kiểu uint8 có tên khác là byte dùng để mô tả mảng dữ liệu thô, chủ yếu liên quan đọc ghi dữ liệu
  • Kiểu int32 có tên khác là rune dùng để thể hiện giá trị 1 ký tự unicode
  • 4 nhóm toán tử gồm:
- Phép số học (+, -, /, %)
- Phép toán bit (&, |, ^, &, <<, >>)
- Phép so sánh (==, !=, <, <=, >, >=)
- Phép toán luận lý (&&, ||)
  • Số thực: float32 và float64, hai kiểu số phức là complex64 và complex128
  • Boolean: true và false

Chuỗi (string)

  • Chứa một chuỗi byte dữ liệu không thay đổi sau khi đã tạo
  • Các kí tự được bọc trong cặp dấu nháy kép hoặc nháy đơn được dùng để biểu diễn văn bản
  • Dùng fmt.Sprintf hoặc strconv.Itoa để chuyển số thành chuỗi. 
var a int = 1234
s1  :=  fmt.Sprintf("%d",a)
fmt.Println(s1) #1234
fmt.Printf("type s1 is %T ", s1) #type s1 is string

Biến

  • Gồm có 2 phần là tên biến và kiểu dữ liệu
  • Dùng từ khóa var, theo sau là tên biến, rồi đến kiểu dữ liệu, cuối cùng gán giá trị cho biến
var x string = "Hello World"

var x string
x  = "Hello World"

x := "Hello World"

Hằng số

  • Là các biến không thể thay đổi dược giá trị, dùng từ khóa const
const x string = "Hello World"

Mảng (Array)

  • Là một tập hợp các phần tử có cùng kiểu dữ liệu nằm liên tiếp nhau, mảng không thay đổi chiều dài được, để truy xuất một phần tử của mảng sử dụng toán tử [], khai báo:
var x [5]int
x := [5]float64{ 98, 93, 77, 82, 83 }

Slice

  • Là kiểu dữ liệu dạng tập hợp như array, các phần tử được đánh chỉ số, khác array ở chỗ số phần tử trong slice có thể thay đổi được, khai báo:
var x []float64 #slice có 0 phần tử
x := make([]float64, 5) #slice có 5 phần tử
x := []int{1, 2, 3} #slice có 3 phần tử kiểu int

arr := [5]float64{98, 93, 77, 82, 83} #tạo slice từ array bằng cách dùng [low:high]
x := arr[0:5] #[low:high] lấy các phần tử từ vị trí low đến vị trí high - 1
  • Hàm hỗ trợ slice là append() và copy()
  • append(slice1, 4, 5) tạo một slice mới có các phần tử giống như slice1 và thêm vào 2 phần tử mang giá trị 4 và 5
  • copy(slice2, slice1) sẽ sao chép các phần tử trong slice1 vào slice2

Map

  • Là kiểu dữ liệu dạng tập hợp nhưng các phần tử trong map không có thứ tự, tức là không thể truy xuất các phần tử theo chỉ số như slice với array
  • Các phần tử trong map là các cặp khóa-giá trị (từ điển), truy xuất các phần tử trong map được thực hiện thông qua khóa.
  • Dùng từ khóa map, sau đó là kiểu dữ liệu của khóa trong cặp dấu ngoặc vuông [], rồi đến kiểu dữ liệu của giá trị.
var x map[string]int
  • Tạo một map bằng cách dùng hàm make
x := make(map[string]int)
  • Gán giá trị và truy xuất giá trị trong map dùng khóa
  • Xóa một phần tử trong map bằng hàm delete()
delete(x, "key") #tên map và khóa cần xóa
  • Khi truy xuất một phần tử của map nhận được 2 giá trị trả về gồm giá trị của khóa và giá trị kiểu boolean cho biết việc truy xuất có thành công hay không (true hoặc false)

Các lệnh điều khiển

  • Gồm for, if, switch

Phân nhánh

  • Lệnh if
if <mô tả 1 (tùy chọn)>; <biểu thức luận lý 1> { 
<Khối lệnh 1> 
} else if <mô tả 2 (tùy chọn)>; <biểu thức luận lý 2> { 
<Khối lệnh 2> 
} else if <mô tả n-1 (tùy chọn)>; <biểu thức luận lý n-1> { 
<Khối lệnh n-1> 
} else { 
<Khối lệnh n> 
}

Switch

  • Là một biến thể của câu lệnh if, dùng trong trường hợp có nhiều phân nhánh
  • Dùng từ khóa switch, theo sau là một biểu thức điều kiện, rồi tới một danh sách các từ khóa case, ứng với mỗi từ khóa case là một giá trị nào đó, rồi tới dấu 2 chấm : và các lệnh sẽ được thực hiện.
switch <mô tả (tùy chọn)>; <biểu thức (tùy chọn)> { 
case <biểu thức 1>: 
    <Khối lệnh 1> 
... 
case <biểu thức n>: 
    <Khối lệnh n> 
default: 
    <Khối lệnh mặc định> 
}
  • Nếu biểu thức điều kiện ở switch cho kết quả trùng với giá trị ở từ khóa case nào thì thực hiện các câu lệnh sau từ khóa case đó
  • Từ khóa default thực hiện các câu lệnh nếu giá trị ở switch không trùng với bất kì từ khóa case nào

Lặp

  • Go chỉ cung cấp vòng lặp for nhưng có nhiều cú pháp khác nhau để đáp ứng yêu cầu đa dạng của các loại vòng lặp, có 4 dạng tương thích while, do while, for và for range
for { // Như vòng lặp do while 
    <Khối lệnh> 
} 
for <Biểu thức luận lý> { // Như vòng lặp while 
    <Khối lệnh> 
} 
for <Mô tả trước>; <Biểu thức luận lý>; <Mô tả sau> { 
    <Khối lệnh> 
} 
for <Chỉ số>, <Giá trị> := range <Biến kiểu dữ liệu có phần tử> { 
    <Khối lệnh> 
}
